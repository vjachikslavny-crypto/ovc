# Design Fixes

- Главная страница `/` стала dashboard-экраном с аккаунтом, недавними заметками, списком и вкладкой графа.
- Редактор вынесен на `/editor`, старые ссылки `/?note_id=...` редиректят на `/notes/{id}`.
- Навигация получила отдельную ссылку `Главная`, а `Редактор` ведет на `/editor`.
- Добавлен `src/static/js/welcome_dashboard.js` для загрузки заметок, графа и создания новой заметки с главной.
- Авторизация стала читать access token из cookie и мягче обновлять локальную сессию.
- Переход из мини-графа изменен на `/notes/{id}`.
- Для аудио waveform добавлен fallback без `audioop`.

- Переработан визуал кнопок редактора: круглые отдельные контролы, мягкий focus без квадратной обводки.
- Убраны лишние подписи в toolbar и inline bubble, текстовые кнопки заменены на компактные символы с `aria-label`/`title`.
- Переработан визуал окна паспорта заметки под кнопкой `i`: больше воздуха, секции-карточки, компактные связи, форма добавления связи не уезжает вниз при большом списке.
- Таблицы в редакторе перестали растягивать страницу вправо при длинном тексте в ячейках.
- Markdown-превью получило внутренний горизонтальный скролл для широких таблиц и code blocks.
- После добавления связи мини-граф принудительно обновляется без перезагрузки страницы.
- Даты создания/обновления в паспорте заметки выводятся нормальным русским форматом вместо сырого ISO.
- Убраны прямоугольные рамки вокруг шапки редактора и активных текстовых блоков.
- Главный экран упрощен до финального вида: убраны служебные подписи, левая и правая зоны выровнены в две равные колонки.
- Удалены исходные CSS-правила, которые рисовали верхнюю полоску карточек и квадратные focus-рамки. Для `styles.css` добавлен cache-buster, чтобы браузер не держал старую версию.
- Убрана рамка всего контейнера редактора, из-за которой верхняя панель выглядела как обведенная прямоугольная коробка.
